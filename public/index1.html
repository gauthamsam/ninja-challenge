Exhaustive Question Types

You name it: descriptive, fill in the blanks, true or false and multiple choice - and it's there. Comprehension support is also included so that you can assess candidates better.

Auto and Manual Evaluation

Zoho Challenge attempts to auto evaluate all answers. However some questions like descriptive questions are best evaluated manually. Zoho Challenge supports manual evaluation for such question types. Of course, you can invite other evaluators to help you out

Quick Quiz Results

Test results are generated on the fly. With configurable pass percentages, you can get results sorted by rank too. You can also configure what your candidates see after submitting a test.

Secure Testing

Apart from regular email and login based testing, you can deliver invigilated tests using supervisor codes. You can also set up a test so that it can be taken only from your office / campus

Who can use Zoho Challenge?

Teachers

Schedule repetitive tests with time limits and monitor student performance in detail - there's rank ordering and separate performance profile view for each student.

Schools

Invite all students as candidates, setup their profile and organize them as "batches". Assign tests to several batches of students in one go. If auto-evaluation doesn't cut it for you, add invite multiple evaluators to scrutinize answers.

Businesses

Screen job applications with timed tests and random questions. Encourage continuous learning. Leverage higher security to control access to assessments.

Study Groups

Prepare tests with questions sourced from multiple question banks. Each member in the study group can contribute questions in their chosen subject. Compare answers against a pre-coded key after every assessment.
